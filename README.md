# Лабараторные работы по математической статистике

**Стек технологий:** Python (pandas, scipy, matplotlib, numpy)


## Установка и запуск
1. **клонировать репозиторий**
```bash
git clone https://github.com/vfhc26/Math-Statistic.git
```
2. **создать виртуальное окружение**
```bash
python -m venv venv
source venv/bin/activate # linux
venv\Scripts\activate    # Windows
```
3. **установить зависимости**
```bash 
pip install -r requirements.txt
```
4. **Запуск лабараторной работы**
```bash
cd *name of laba directory*
python main.py
```
